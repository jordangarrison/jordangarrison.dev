# jordangarrison.dev

Personal portfolio website built with [Astro](https://astro.build), [Tailwind CSS](https://tailwindcss.com), and [Rose Pine](https://rosepinetheme.com) theming.

## Development

```bash
# Enter Nix dev shell
nix develop

# Install dependencies
bun install

# Start dev server
bun run dev

# Build
bun run build
```
